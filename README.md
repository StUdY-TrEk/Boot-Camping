# 🖥️ study Trek
- 프로젝트명 : study Trek
- 한 줄 정리 :  부트캠프 리뷰 및 Q&A 플랫폼
- 내용 : 다양한 부트캠프에 대한 수료자 리뷰 제공 및 예비 수강생과 현재 수강생의 질문에 대한 수료자의 답변을 통해 부트캠프 선택 및 학습 과정 지원
  
  
# 🎓 Tech Stack
- 언어 : Java
- 버전 : JDK17
- Tools : GitHub, Git
- IDE : IntelliJ IDEA
- DB: MySQL 8.0.37
- Framework : SpringBoot 3.2.5
- 인증/인가 방식 : JWT
